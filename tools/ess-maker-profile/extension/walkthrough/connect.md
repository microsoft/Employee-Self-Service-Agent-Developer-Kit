# Connect

The **Connect** button signs you in to your Power Platform environment so the kit can:

- Discover your deployed ESS agent and its components.
- Create a local working copy for safe editing.
- Validate connectivity before any changes are pushed.

When you click Connect, a chat opens with the `/setup` command — just answer the prompts (environment URL, then sign-in).

> First time? You'll see a browser pop-up asking you to sign in with your work account. That's expected.
