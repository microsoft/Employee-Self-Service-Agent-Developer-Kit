# Welcome

You're about to customize your Employee Self-Service agent — the assistant your employees use for HR, IT and facilities requests.

## How this works

1. **You describe** what you want in plain English.
2. **GitHub Copilot** does the heavy lifting — generating topic configuration, integration wiring, adaptive cards and test sets.
3. **The kit** safely deploys your changes: checkpoint → preview → push → verify. You can always roll back.

You don't need to know YAML, JSON or any code. Work through the steps on the left.
