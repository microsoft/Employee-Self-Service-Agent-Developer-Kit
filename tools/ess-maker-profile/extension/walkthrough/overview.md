# How the kit works

You're about to customize your Employee Self-Service agent — the assistant your employees use for HR, IT and facilities requests.

## The workflow

1. **Connect** — Sign in to your Power Platform environment.
2. **Create a topic** — Describe what you want in plain English. The kit generates everything.
3. **Scan** — Check for broken references and configuration issues.
4. **FlightCheck** — Run 41+ automated readiness checks.
5. **Push** — Safely deploy your changes to Copilot Studio.

Each step is available as a button in the **Quick Actions** panel on the left. Click any button to open a guided chat — just answer the prompts.

You don't need to know YAML, JSON or any code.
